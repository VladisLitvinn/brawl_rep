# Brawl Stars — Симулятор боксов

Браузерный симулятор открытия боксов с авторизацией, жетонами и таблицей лидеров.

## Быстрый деплой в Git (с контейнером)

1. **Инициализация и первый коммит** (если репозитория ещё нет):

```bash
git init
git add .
git commit -m "Initial: симулятор боксов + Docker"
```

2. **Создайте репозиторий** на GitHub/GitLab (без README, .gitignore — они уже в проекте).

3. **Подключите remote и запушьте**:

```bash
git remote add origin https://github.com/ВАШ_ЛОГИН/brawl_stars_boxes_simulator.git
git branch -M main
git push -u origin main
```

4. **Запуск на любой машине** (клонирование + контейнер):

```bash
git clone https://github.com/ВАШ_ЛОГИН/brawl_stars_boxes_simulator.git
cd brawl_stars_boxes_simulator
docker compose up --build -d
# Приложение: http://localhost:8888
```

Репозиторий уже содержит `Dockerfile` и `docker-compose.yml`, поэтому после `git clone` достаточно выполнить `docker compose up --build`.

## Запуск через Docker

```bash
docker compose up --build
# Откройте http://localhost:8888
```

Поднимаются сервисы:
- **PostgreSQL 15** — БД пользователей и полученных бравлеров (порт 5432).
- **Приложение** — Node.js-сервер (порт 8888): раздача фронта и API.

При первом запуске создаются таблицы `users` и `user_brawlers`, а также добавляются тестовые пользователи (если их ещё нет). Новым пользователям при регистрации начисляется 100 жетонов.

**Тестовые пользователи** (пароль у всех: `123456`):
- **Админ** — администратор (в интерфейсе отображается метка «Админ»).
- **Иван**, **Мария**, **Алексей**, **Дмитрий**, **Елена**, **Николай** — обычные пользователи с разным количеством полученных бравлеров (разный рейтинг в таблице лидеров).

**Полная пересборка БД** (очистить все данные и поднять заново):
```bash
docker compose down -v
docker compose up --build -d
```
Флаг `-v` удаляет том `pgdata` с данными PostgreSQL; при следующем `up` БД создаётся заново, таблицы и тестовые пользователи — по новой.

## Возможности

- **Вход / регистрация** — без входа симулятор недоступен.
- **Жетоны** — отображаются в шапке. Обычный ящик — 9 жетонов, большой — 18. Без достаточного количества жетонов открытие недоступно.
- **Симулятор** — открытие ящиков, выпадение бравлера и (с шансом) «Пропуск 1 ДЗ».
- **Мои бравлеры** — список всех полученных бравлеров.
- **Лидеры** — таблица игроков по количеству полученных бравлеров.
- **Шансы** — страница с вероятностями.
- **Админ** (только для пользователя с меткой «Админ») — управление пользователями: добавление/удаление пользователей, изменение количества жетонов у любого пользователя (добавить/убрать или установить точное значение).

Иконки бравлеров и ящиков — локальные (`assets/`), скачаны с [Brawlify CDN](https://cdn.brawlify.com/). При обновлении `brawlers.json` запустите `npm run download-assets`. Шансы редкостей — по официальным данным Supercell (legacy Brawl Box / Mega Box).
